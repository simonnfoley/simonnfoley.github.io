Mobile Autonomic Security for Network Access Controls

Oct 14, 2016 Simon Foley

MASON is an agent that runs on an Android device and uses a knowledge base to automatically and in real-time configure the Android iptables firewall. The agent acts as an automated security expert, managing security on the device, on behalf of a non-expert user. Ultan Neville implemented the current prototype which requires a rooted Android device with a kernel that supports Netfilter/iptables. Mason currently monitors six attributes of the Android state and configures iptables according to best-practices from NIST 800-41/ NIST 800-41rev1, NIST 800-124, NIST 800-114 and NIST 800-153. Further details on the design and operation of Mason can be found here.
